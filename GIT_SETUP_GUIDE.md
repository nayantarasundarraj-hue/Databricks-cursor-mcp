## ğŸ› Common Issues

### "Permission denied (publickey)"

Use HTTPS instead of SSH:
```bash
git remote set-url origin https://github.com/YOUR-USERNAME/databricks-mcp-server.git
```

### "Updates were rejected"

Someone else pushed first:
```bash
git pull --rebase
git push
```

### "Node_modules committed by mistake"

```bash
# Remove from git but keep locally
git rm -r --cached node_modules
git commit -m "Remove node_modules from git"
git push
```

### "Committed credentials by accident"

```bash
# Remove from latest commit
git rm --cached .databrickscfg
git commit --amend
git push --force

# If already pushed: rotate credentials!
databricks auth login --profile dev
```
